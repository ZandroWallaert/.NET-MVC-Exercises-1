# Lab Vierkantsvergelijking

This lab will teach you how to create a new console app from scratch with your own business logic. In your code you will learn to work with nullable value types, getters, setters and the System.Math namespace.

## Exercise

Create a console App to solve a "Vierkantsvegelijking".
[https://nl.wikipedia.org/wiki/Vierkantsvergelijking](https://nl.wikipedia.org/wiki/Vierkantsvergelijking)

- Create setters and getters voor A, B and C.
- Create getters for Discriminant, X1 and X2.
- When the Discriminant < 0 then return for X1 and X2 both null.
_([Nullable value types](https://docs.microsoft.com/en-us/dotnet/csharp/language-reference/builtin-types/nullable-value-types))_

Example:
```bash
A:
1
B:
4
C:
-5
1xÂ² + 4x + -5 = 0
Discriminant:  36
X1: 1
X2: -5
```
